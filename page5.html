Project Summary and Acknowledgments

Project Overview

In this showcase, Nirmal and Pradeep have developed three engaging projects to demonstrate the versatility and power of simple web technologies: HTML, CSS, and JavaScript.

  • Rock Paper Scissors Game: A classic game implemented using HTML buttons for user input, CSS for styling, and JavaScript for game logic.
  • Coin Toss Game: A fun game where the user can flip a virtual coin and see the result, built with HTML, CSS, and JavaScript.
  • To-Do List: A functional to-do list application that allows users to add and delete tasks, using HTML, CSS, and JavaScript.
